# Flip-Flops and Counters Project

## Objective
To study and implement various flip-flops and counters used in digital electronics.

## Contents
- SR Flip-Flop
- JK Flip-Flop
- D Flip-Flop
- T Flip-Flop
- Ripple Counter
- Synchronous Counter
- Mod-10 Counter

## Tools Used
- Logisim
- Proteus
- Verilog
- GitHub

## Results
Successfully simulated flip-flops and counters and verified output waveforms.
